# Desafío técnico Nubi

## Instrucciones para iniciar

Clonar el respositorio e instalar dependencias:

### `git clone https://github.com/lucastl/app-clima.git`
### `cd app-clima`
### `yarn install`

Iniciar aplicación:

### `yarn start`
